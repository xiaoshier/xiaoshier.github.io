2016 年总结

发表于 2017-01-15 | 分类于 life
2016年总结

2016 年总结

  1. 工作篇

    15年的工作重心是与 html 和 css 做斗争, 年后离职后, 发现 js 才是王道, 而且功能很强大, 于是在16年, 入职的公司, 使用了 angularjs, 在对 js 了解泛泛的情况下,
    就边学便用了 angularjs, 对自己能有这个勇气, 都感到佩服, 从开始工作之后, 一切都是现用现学, 没有一个人带着, 而且也是挺乐于折腾, 一直在提升自己的能力, 当然
    能有这个意识, 也是受姐夫影响, 在这里谢谢他.

    16年工作了几个月, 年后3月底, 清明节前离职, 五一过后入职工作一周, 六月份端午节后入职, 工作了两个月, 巧妙的把所有节假日在家度过, 现在想想, 挺感谢最后一家
    公司, 在这里也可以说是尽心尽责, 写的东西, 绝对比他们之前的有质量, 也感谢他们给我提供了一个可以任我支配的平台, 让我接触到并使用了 angularjs, 可以说是我
    又迈出了一大步. 从他们公司离职后, 回家读了 underscorejs 的源码, 对 js 有了进一步的了解, 试想一下, 如果我还继续在那里工作, 只是增加了我搬砖的经验, 没有
    了解砖是怎么制作的机会. 一切皆有定数.

    16年年底, 也就是找工作不顺利, 在家闲着的时候, 开始着手写博客, 意识中知道的东西, 和用手写出来的东西, 还是有很大的区别的, 用手写出来, 能更加清晰的知道哪里懂了,
    哪里还有待查证, 了解. 用文字记录一下自己的生活, 工作, 成长, 挺不错.

    搭建 git 博客, 使用 markdown 写博客, 使用 git 版本管理, 把 git 和 markdown 的基本操作也都掌握了, 不错不错. 要想学会一个东西, 那就是先用起来, 再去深入了解.

    还是同样的原因, 闲时研究了一下 vim 编辑器, 终于通过不懈努力, 掌握了满足正常工作需要的功能, 对于这个还是比较满意的, 受姐夫影响, 比较喜欢用快捷键, 电脑操作
    的快捷键, 编辑器的快捷键, 浏览器使用的快捷键等等.

    回家做了三个项目的兼职, 只有一个项目给了钱, 其中一个项目, 联系人说黄了, 另一个项目就是简单的支持了一下, 让我知道了, 做兼职还真是不如上班, 有了此经历, 又丰富
    了我的人生, 说人生有点大, 嘻嘻

    学习一种技能, 先模仿, 后深入; 先用起来, 再往好了用

  2. 生活篇

    一上班, 就有精神头, 也有控制力, 有了精神依托把, 体重一度减到63kg, 但自打在家赋闲时, 体重就又飙上来了, 一度飙到68kg, 现在每天做瑜伽, 俯卧撑, 努力减肥中,
    人活着是要有精神依托的, 是要有底线的, 一个人连自己的体重都管理不好, 那说明这个人得自制能力有问题, 这句话说得不错, 没有天生的胖子, 只有管不住自己的胖子,
    何况也有减肥成功的经历, 一度减到56kg, 由于后面要上课, 要工作, 就放弃了, 不后悔, 继续努力就好.

  3. 感情篇

    自2014年10月份与姐姐姐夫一家人住了有两年又一个月(11月份回了自己的家).

    由于小时候一是父母忙于生计, 无暇顾及我们, 二是他们文化程度不高, 也无意识顾忌在情感方面对我们的培养, 就导致我情感方面比较冷漠, 不知亲情为何物,
    小时候太小, 与姐姐也没什么精神方面的沟通, 等到初中之后, 姐姐在外地上学, 从此之后也就一直与她两地分隔, 相处的时间, 也就只有寒暑假, 更是没机会深入沟通,
    导致我现在都习惯把心事藏在心里, 几乎没和人沟通过.

    在姐姐姐夫家两年的时间里, 与姐夫沟通过不少次数, 都是在他感觉我处于要崩溃的状态, 给我进行疏导, 我的习惯是听的多于说的, 有很多观念都是由此产生或者改变,
    有句话说的好, 授人以鱼不如授人以渔, 有人在物质上给予你极大的满足, 不如在精神层次对你进行帮助, 在这里得感谢他.

    刚到北京, 姐姐就怀有小宝宝了, 算是陪着她一起度过孕期产期, 怀孕期间, 姐姐变得很可爱, 无忧无虑, 过得很开心, 我看着她开心, 我也蛮高兴.

    我工作一个月后, 小外甥就诞生了, 看着小小的一坨肉呼呼的, 刚开始都不敢抱他, 生怕在我手里化掉了, 抱着他的感觉很好, 软乎乎的趴在你的肩头,
    但是我情感方面的缺失, 对他没有过多想象中应该有的种种感觉, 略感遗憾, 不过我已经意识到这方面的问题, 也在积极调整中, 这将是改变对待人得方式, 态度.

    与外甥, 外甥爷爷奶奶, 一家人一起生活, 无形中对我造成了一些影响, 原来孩子是用来宠的, 爷爷奶奶也对我也很好, 大家都在努力付出, 维持家庭的和谐,
    爷爷奶奶堪称史上最好公公婆婆, 尤其是奶奶在看到我的整套开线之后, 给我缝上了, 这个着实让我感动不已, 话说我这辈子真心感动的时候还真不多,
    但是在一次宝宝感冒很久都好, 奶奶他们一直认为生病了去大医院检查, 去小诊所拿药, 奶奶看着宝宝天天喝特别苦的药, 还一直没好转, 便私自给宝宝吃了小诊所的药,
    在一次姑姑来家里, 奶奶跟我一起睡的时候, 给我说了, 我当时挺震惊, 说这个必须要让姐姐姐夫知道, 然后就开始有了列横, 加上我与他们相处习惯了之后, 就暴露了本性,
    对方说话, 如果我当时不开心, 或者不想说话时, 我基本上是保持沉默, 这是非常不礼貌的(以后一定要改掉, 因为我固有的思维是, 有些话我不说, 他也是知道的, 开什么玩笑,
    你在想什么, 别人怎么知道, 要改, 一定要改).

    由于已经是十一月份, 找工作也不太顺利, 就干脆回家了. 在家里的这段时间, 也是感触颇深.

    刚回家的那段时间, 由于父母之前一直住在平房, 而且干农活, 已经很累了, 没有习惯打扫卫生, 收拾屋子, 现在在楼房上, 经常是我这边打扫, 一顿饭的功夫, 真真的是一顿饭吃完,
    地上就脏成一片, 我经常冲他们发火, 大吼大叫, 沙发上不能放衣服, 厨房做完饭要收拾整齐, 擦干净等等, 尤其是早晨刚起床, 起床气加上看见家里乱七八糟, 更是少不了一通吼叫,
    在持续了一段时间后, 他们稍有改善, 现在是我每天去图书馆学习, 他们在家里, 我也不太在意房间变成什么样, 在我不管不顾的这段时间里, 变得好些了, 后来想明白了,
    这就是他们几十年固有的生活方式, 不是一时半会就能改变的, 而且把我的想法, 我的习惯强加于他们身上也是不合理的, 于是乎我就找到了我们相处的平衡点, 我不再会因为看见房间
    乱七八糟而冲他们大吼大叫, 我只打扫我的房间, 厨房偶尔早起做早饭时, 顺便把厨房规整一下, 此种相处模式, 大家都好.

    刚在家时, 还是北京的作息时间, 早起就顺便把粥熬上, 老爸老妈非常不习惯我变得如此勤快, 竟然不是他们印象中我应该是睡到日上三竿, 叫 n 遍才起床, 他们很开心, 也很不适应,
    父母需要很简单, 偶尔做一顿早饭, 都能让他们觉得开心, 你怎么起这么早, 哟, 孩子做早饭呢. 愧疚哇, 现在是原形毕露了.

    老爸习惯给我说他认识人的一些事, 往往都是被我打断, 我不认识你说的人, 我不想听, 其实他也是想跟我说说话, 但是我们之间又没有其他的共同语言, 只能说些他知道的人和事, 这个我
    也知道, 但就是耐不下心来听他絮叨, 以后尽量耐着性子听一听吧, 如果父母百年后, 我一定会怀念他们在我耳边说些我不知道的人和事.

    老妈习惯在看电视时, 充当解说员, 也被我狠狠的纠正, 今天电视里播放<<神探狄仁杰 第一部>>, 元芳与人对抗, 地方从沙漠下面飞了出来, 老妈说:’咦, 快看, 这么厉害, 从沙漠下面
    飞出来的…’, 我就没好气的说, 这部电视剧, 咱们前一段时间才看过好不好, 至于这么惊讶么, 想来也搞笑. 我喜欢在看电影或电视剧的时候听别人剧透, 是不是受老妈影响, 但我却不
    愿意听她说话, 唉, 这女儿当的太不称职了.

    老妈自入冬以来, 一直在做鞋子, 拖鞋, 表姐家孩子的鞋, 由于姐姐他们在北京, 没人穿做的布鞋, 在老妈多次问询, 不需要她做鞋后, 就给表姐家孩子做了两双鞋, 现在在给老爸做布鞋中,
    我昨天突然想着, 让老妈也给我做一双, 不穿, 在那里放着, 等她百年后, 也能有个念想, 但我不敢这么给她说, 找她给我做一双鞋, 结果人家不干, 说法是你都该伺候我给我做鞋了, 还让我
    再给你做, 别想了, 考虑到老妈糖尿病并发症, 手指头疼, 就没坚持. 不知道以后会不会后悔.

    还有就是爸妈耳朵不好使了, 看电视声音堪比用了音响, 每次我要睡觉了, 就把音量调到我听着都小, 他们肯定听不到的程度, 没办法, 谁让他们遇到了一个无赖女儿呢,
    他们心里肯定说:”忍字心头一把刀, 我忍”, 哈哈. 后面要考虑给他们安装助听器了.

    最近一直在家吃过早饭, 然后就去图书馆, 在外面吃一顿午饭, 减肥中, 晚饭不吃. 前天早晨老爸突然炖了羊排, 他们出去跳舞前, 给我说, 锅里炖了羊排, 吃了再走, 一定是他们想让我吃些
    好吃的, 但白天晚上又不在家, 只能放到早晨了, 等他们回来, 我就开玩笑说:” 无事献殷勤, 非奸即盗, 老爸, 你说, 有什么要求, 说出来”, 把老爸逗乐了. 我甚至一度怀疑他们买的羊肉是
    别人的处理货 (即非正常死亡, 便宜处理), 怎么就这么大方, 舍得买这么多羊肉吃, 我也向他们提出了我的质疑, 但是通过他们的回答以及对他们的表情观察, 看不出什么破绽来, 谁让我是
    一旦知道那些肉食不是正常物件时, 宁愿饿着不吃的主(大家都是这样, 但我爸妈不是如此). 小时候, 家境很不好, 能吃个鸡蛋, 都高兴半天, 家里养的猪, 不知什么原因死掉了, 被我妈全部
    油炸了, 我记忆中, 是一口没吃, 宁愿馋着, 我也吃不下, 就是过不了心里这一关, 嗓子眼不舒服.

    老爸对老妈是又恨又爱吧, 极度依赖老妈, 但对老妈对他们家人的态度极度不满, 不论大事小事, 都会询问老妈, 小到做饭放什么, 怎么做, 这都要问, 受不了. 但对老妈那叫好, 给她按摩,
    一按按半天, 搁在我身上肯定是不会做的, 小时候屈于他们的压迫(其实也没按过几次), 会在不情愿的情况下按两下, 现在直接说, 老爸赶紧伺候着.

    爸妈现在是老咯, 跟自己女儿说话, 都得看着我的表情…

    孩子一定是父母上辈子的债权人, 这辈子是向他们来讨债的, 任你对他们冷言冷语, 大吼大叫, 他们虽一时气愤, 但你永远使他们的孩子, 无论你做的多么的过分, 他们还是会包容你, 原谅你
    这辈子没有任何人能做到他们这样, 这也是此次回家感悟到的, 对我以往的种种恶略行径进行反思, 以后一定要对父母好些, 他们其实也不求孩子能怎么样, 跟他们开开玩笑, 说说话就够了,
    但千万别想着去改变他们这么多年形成的消费观, 人生观, 价值观, 这样只会两败俱伤, 让你觉得他们怎么这么不可理喻, 让他们觉得我的孩子怎么这么不孝, 处处跟我作对, 最好的方式,
    就是你按照你的方式, 过你自己的生活, 他们按照他们的方式, 过他们自己的生活, 也许在你眼里, 他们是在受苦等等, 但他们不仅是习惯了, 而且认为是正确的, 这样他们过得开心, 安心.

    不管身处何处, 与何人在一起, 不要忘了, 父母是这个世界上唯一能无限度包容你, 想让你过得好的人. 谢谢爸妈, 虽然我一度觉得你们对我不好, 只管生, 不管养, 这是我自私了, 你们何尝
    不也是这么过来的, 我的条件相较于你们好了多少倍, 能上学, 能不挨饿, 有更好的物质条件.

    此刻需要我有意识的纠正自己的错误, 有意识的培养对情感的认知, 有意识的对他们好, 来回报他们的养育之恩.
