bash 常用操作

发表于 2017-02-24 | 分类于 bash

自从使用 Mac 之后, 少不了要使用终端, 使用命令行进行一些操作, 从刚开始的哇塞, 这都是什么这么神奇, 到后面需要安装一些软件等, 基本上都是按照网站上的提示进行操作, 今天得空, 在网上搜索了一下 bash 操作, 收获颇丰啊, 现在把一些常用的命令整理一下, 方便日后查阅.

item2 常用快捷键

快捷键 功能
Command + t 新建标签
Command + w 关闭标签
Command + Number 切换标签
Command + enter 全屏切换
Command + f 查找
Command + d 垂直分屏
Command + shift + d 水平分屏
Command + [/] 切换同一标签中的屏幕
Command + shift + h 查看剪切板历史
Command + ; 查看当前目录下历史命令

bash 常用操作快捷键

光标移动

快捷键 功能
Ctrl + a 移至命令行首 ahead
Ctrl + e 移至命令行尾 end
Ctrl + f 按字符前移(向左移动) forward
Ctrl + b 按字符后移(向右移动) back
(Double Click) Ctrl + x 行首到当前光标切换

编辑

快捷键 功能
Ctrl + w 删除当期光标临近左边单词 word
Ctrl + h 删除光标前一个字符
Ctrl + d 删除光标后一个字符 delete
Ctrl + u 删除光标左边所有
Ctrl + k 删除光标右边所有
Ctrl + l / Command + r 清屏
Ctrl + / 撤销上一步操作
Ctrl + y 粘贴删除的第一项

命令记录

快捷键 功能
Ctrl + n 下一行命令 next
Ctrl + p 上一行命令 previous
Ctrl + r 向后增量搜索历史命令
Ctrl + o 循环执行历史命令
