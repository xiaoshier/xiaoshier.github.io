中央电视台 — 视听资料TSCU定价系统项目

PC 端 web 页面开发，这是一个影视资料，图片等定价出售的一个平台。
入行的第一个项目，在只是针对 html，css 做过练习未在其他浏览器进行兼容尝试的前提下， 实际做这个项目时，遇到了如 IE8 及以下浏览器兼容 css3 新属性如圆角，透明背景，输入框不显示占位符，以兼容模式渲染页面； Chrome 浏览器中输入框有蓝色外边框，字体小于10px 时以 10px 显示，苹果设备中浏览器图片不清晰等等问题。 通过看书以及网上查找解决方案，让我对不同设备以及浏览器间的差异有了一定的认识， 如苹果 retina 屏设备具备高像素密度，会导致一般屏幕下清晰的图片在苹果设备下会模糊，需要采取针对苹果设备加载大尺寸图片的解决办法； 后端在套页面时，某些页面遗漏了 DOCTYPE 声明语句，导致其在 IE 浏览器中以兼容模式显示，布局发生错乱，针对这个问题，排查了半天，发现少了该声明语句，通过这个问题，不仅知道了 DOCTYPE 的作用，也有必要将开发中的注意事项以文档的形式告知相关同事。
通过这个项目学会如何定位问题原因，如何和后端同事沟通配合，也认识到需要去了解 W3C 文档，了解每一个元素，每一个属性是如何定义，如何使用。
