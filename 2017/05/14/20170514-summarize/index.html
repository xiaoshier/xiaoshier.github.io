3-5月份工作小结

发表于 2017-05-14

对最近的工作做一个简要总结.

工作的一些事

至此时入职新公司两个月时间, 这是从事前端行业以来, 算是最忙碌的, 所涉及的业务也是最复杂的一份工作了, 之前的工作经验主要是以实现页面且是单人工作为主, 这次是多人合作, 直播 app 混合式开发, 以及微信端的观看直播功能.

刚入职时由于没有文档, 相关负责人给我把项目安装了, 可以运行, 简单的告诉我正常流程的操作步骤, 一些配置文件直接拷贝给我(如 nginx), 帮我把 git 配置配好可用.

项目是以微信作为平台, 登录信息是通过微信获取的, 本地开发时需要把浏览器改为微信内核, 以及定期更新用户信息, 由于刚来是一个小伙伴把他本地的代码直接拷贝给我的, 他把本地开发的相关操作做了, 我的项目是可以启动的, 就没有告知我需要把用户信息拷贝到项目中用作登录使用等相关信息, 就经常出现项目无法正常启动等问题, 后面是自己一点点问出来琢磨出来的, 也是蛮痛苦的, 后面就梳理了一下文档, 并实时更新文档, 在梳理的过程中, 对项目是越来越了解, 以及越来越清晰. 古人云: 好记性不如烂笔头, 太对了. 也通过如此, 锻炼了我写文档的技能, 如何清晰有条理的整理内容, 如何排版, 如何分类等等.

项目安装时是直接把已有的 nginx 配置文件拷贝至我本地的, 本机访问正常, 但是很不巧的是, 我用 Mac 开发, Mac 防火墙开启使得手机无法访问到 nginx 启的项目, 不得已又对 nginx 的配置做以了解. 手机无法访问项目起初定位是 nginx 配置文件有问题, 在对 nginx 配置学习了解之后仍未解决, 最后定位到 Mac 的防火墙在捣乱. 歪打正着的是后面又启了新项目, 这块儿的配置很轻松搞定.

把 gerrit 的使用也简单了解了下. 这个是项目负责人给我设置好的, 我私下里还是对它研究了一番, 并且也做了文档形式的记录. 又歪打正着, 后面服务器崩了, 又重新搭建了服务器, 该项操作得重新来一遍, 我之前做过功课, 但未实际操作过, 这次刚好给了一次实践的机会, 刚好查漏补缺, 使用流程算是全部搞懂了.

最重要的是 git 的操作, 之前对 git 的使用, 仅仅是简单的 clone 代码, push 到 github 上. git 的操作也是遇见诸多问题, 如提交后发现有问题要改, 取消提交, 修改后再次提交, 出现了多个 commitId 无法合并入库, 本地版本回退等等问题, 也是通过私下里去学习, 加上实际操作, 才能够说是真正的掌握了该技术.

发现在学习, 以及实际操作中, 能及时的做笔记整理文档, 是非常重要的一件事, 因为当前出了问题, 花了时间解决了, 而且是搞明白那种, 结果时间一长就会忘记了, 再次出现问题时, 只能悔恨当初为什么没整理出来呢.

生活中的一些事

今年算是突然开窍了, 感觉是某位得道高僧对我施以法术达到了醍醐灌顶的效果.

  1. 变得爱动脑思考问题了, 之前只是按照习惯, 潜意识的去说话办事; 今年能够去有意识去思考, 我这么说合理否, 听者是什么感受, 他能听明白我想阐述的内容么, 我这个表达方式他能接受么等等, 会站到对方角度去考虑问题, 而且发现一旦站到对方角度去考虑问题, 于人于己都会方便省事很多.

  2. 变得善谈了, 以前总是听别人说, 自己很少发表意见, 因为不走心, 不知道要怎么接话, 要怎么去说, 别人在提问说话时, 我的大脑是空白的, 往往是逼着自己去想, 你一定要想出来点什么, 要说点什么, 结果是什么也说不出来, 索性就习惯性的听别人说了. 现在不是了, 能在别人说话时, 主动思考, 虽然还是不能做到对答如流, 尤其是在比较正式人比较多的场合, 但对自己的进步还是比较满意的, 继续加油努力;

  3. 今年给自己定了个规矩, 想做一件事超过三次, 那就去做, 不要因为考虑结果而畏畏缩缩犹犹豫豫, 去做了, 大不了就是做坏了, 没有什么是不可逆的不可弥补的, 不要让自己心生后悔;

  4. 在看了<<断舍离>>一书, 翻了几页, 明确了不喜欢的东西, 不想要的事情, 不要因为一些物质方面或其他方面的问题, 心生不舍; 不喜欢就扔了, 不要心里不喜欢而脑袋在告诉自己不能扔, 扔了多可惜, 让心与脑打架, 产生内耗. 比如在一时冲动买了一件不是特别喜欢特别满意的衣服, 后面在看到它时会后悔, 会心生厌恶, 最后决定把它丢弃, 因为一件物品让自己心生烦恼了, 那为何还要折磨自己呢. 这个也就促使我再买东西时会仔细考虑, 到底喜欢与否, 反而节省了金钱与精力;

  5. 在处理人际关系时, 更为简单, 那就是真诚待人, 把自己的想法及时告知对方, 不要因为表达理解方面的分歧引起误会; 不晓得周边人感受如何, 反正我的感受是轻松高效了; 实实在在做人.依旧是能够站在对方角度去考虑问题, 人际关系也会因此变得更易掌握.

  6. 把学习英语提上了日程, 并在实践中, 一直在想不如立马动手去做.

  7. 学着使用文字记录自己的感受与想法, 从刚开始的话不成句, 想哪里写哪里, 到现在能有一个写东西的思路, 逻辑性; 不过还是有待加强, 有些东西不是你想想就能获得就能提升的, 确确实实是要去做的; 自认为不是聪明的人, 那就勤能补拙吧, 别人一听就懂就会, 我不行, 那是不是可以学两遍学三遍呢, 不学就永远都不会.

  8. 在心里发生改变的同时, 也对身体进行了一番改造, 采取轻断食的减肥方法, 成功的减到健康体重, 饮食方面也开始注重养生了, 发现自己可以不用吃那么多东西的, 之前只是为了满足自己吃的欲望而吃, 现在整个人都感觉身轻如燕, 也自信多了. 于此同时也开始改变自己的穿衣风格, 从原来的不修边幅(干净整洁还是保证了的), 不考虑搭配, 整天以运动休闲装扮自己, 到现在尝试裙子, 高跟鞋, 化妆品等等, 这些改变也能够愉悦到自己, 喜欢自己的改变.
