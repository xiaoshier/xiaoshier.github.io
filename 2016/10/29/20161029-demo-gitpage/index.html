使用git展示静态页面

发表于 2016-10-29 | 分类于 git

在 git 上展示 demo。只有把项目放在 gh-pages 分支下，才能显示页面。

设备: MacBook Pro
版本: OS X EI Capitan 10.11.5

创建步骤

  1. 先创建仓库 repo-name，将文件上传至该 repo-name 的 master 中
  2. 再创建 gh-pages 分支
  3. 将文件上传至 gh-pages 分支
  4. 浏览器中输入 username.github.io/repo-name
